# HA9001 - Put split list items on separate lines

Sources: [`HangingListClosingParenAnalyzer`](../../packages/analyzers/src/HangingListClosingParenAnalyzer.cs), [`HangingListClosingParenCodeFixProvider`](../../packages/analyzers/src/HangingListClosingParenCodeFixProvider.cs)

## Description

Report argument and parameter lists that leave one item beside the opening parenthesis while another item starts on a later line.

## Motivation

Mixed single-line and multiline list layout is hard to scan and becomes noisy when more items are added. Keeping each item on its own line makes future edits smaller and clearer.

## How to fix violations

Move every item in the wrapped list to its own line and place the closing parenthesis on its own aligned line.

## Code with Diagnostic

```csharp
CallTarget(1,
	2);
```

## Code with Fix

```csharp
CallTarget(
	1,
	2
);
```
