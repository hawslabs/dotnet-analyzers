# HA9006 - Align hanging-list items with continuation indentation

Sources: [`HangingListClosingParenAnalyzer`](../../packages/analyzers/src/HangingListClosingParenAnalyzer.cs), [`HangingListClosingParenCodeFixProvider`](../../packages/analyzers/src/HangingListClosingParenCodeFixProvider.cs)

## Description

Report multiline argument lists whose items all start after the opening parenthesis but are indented more or less than one indentation level past the line containing that opening parenthesis.

## Motivation

Hanging argument lists should form a predictable block under the call line. This keeps expression-bodied members and chained calls readable when arguments wrap onto their own lines.

## How to fix violations

Align each argument one indentation level past the line that contains the opening parenthesis. Keep the closing parenthesis aligned with the opening line.
