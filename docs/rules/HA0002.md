# HA0002 - Format multiline raw string literal indentation

Sources: [`ReturnRawStringLiteralAnalyzer`](../../packages/analyzers/ReturnRawStringLiteralAnalyzer.cs), [`ReturnRawStringLiteralCodeFixProvider`](../../packages/analyzers/ReturnRawStringLiteralCodeFixProvider.cs)

| Topic             | Value                                   |
| :---------------- | :-------------------------------------- |
| Id                | `HA0002`                                |
| Severity          | Warning                                 |
| Enabled           | True                                    |
| Category          | Formatting                              |
| Code fix          | Yes                                     |
| Analyzer          | `ReturnRawStringLiteralAnalyzer`        |
| Code fix provider | `ReturnRawStringLiteralCodeFixProvider` |

## Description

Report multiline raw string literals whose closing delimiter is not indented with the opening delimiter.

For raw string literals returned directly from a method, HA0002 also reports when the opening delimiter starts on the same line as `return`, or when the opening and closing delimiters are not indented one level deeper than the `return` statement.

The rule supports regular raw string literals and interpolated raw string literals.

## Motivation

Multiline raw strings are easier to read when their delimiter indentation matches the surrounding code. Aligning the raw string delimiters with the literal content keeps generated code, JSON, SQL, and other embedded text blocks predictable while preserving the runtime string value.

## How to fix violations

Indent the raw string content and closing delimiter to match the opening delimiter.

For direct return statements, move the raw string opening delimiter to the line after `return`, indent it one level deeper than the return statement, and apply the same base indentation to the raw string content and closing delimiter.

The code fix reindents the raw string source while preserving the string value.

## Code with Diagnostic

### Interpolated raw string

```csharp
public static string InType(string members) {
	return $$"""
using System;

internal static class GeneratedCode {
{{members}}
}
""";
}
```

### Raw string

```csharp
public static string Create() {
	return """
{
	"value": true
}
""";
}
```

### Raw string argument

```csharp
var members = JoinBlocks(
	$$"""
private static void Test() {
{{FormatBlock(body, 1)}}
}
"""
);
```

## Code with Fix

### Interpolated raw string

```csharp
public static string InType(string members) {
	return
		$$"""
		using System;

		internal static class GeneratedCode {
		{{members}}
		}
		""";
}
```

### Raw string

```csharp
public static string Create() {
	return
		"""
		{
			"value": true
		}
		""";
}
```

### Raw string argument

```csharp
var members = JoinBlocks(
	$$"""
	private static void Test() {
	{{FormatBlock(body, 1)}}
	}
	"""
);
```
