# HA9000 - Put hanging-list closing parenthesis on its own line

## Description

Report hanging multiline argument lists, parameter lists, and `while` conditions whose closing parenthesis is not on its own line aligned with the line that contains the opening parenthesis.

HA9000 also covers hanging-list wrappers around multiline raw string literals when the surrounding closing parenthesis needs to move while the raw string indentation is preserved.

## Motivation

Hanging multiline lists are easier to scan when the closing parenthesis visually pairs with the opening line. This keeps nested calls and declarations predictable during reviews.

## How to fix violations

Put the closing parenthesis on its own line and indent it to the same level as the line that contains the opening parenthesis.

## Code with Diagnostic

```csharp
CallTarget(
	1,
	2);
```

## Code with Fix

```csharp
CallTarget(
	1,
	2
);
```
