# HA0003 - Keep 1TBS brace settings consistent

## Description

Report C# files whose effective `.editorconfig` sets `indent_brace_style` to `1TBS` or `OTBS` while a related C# formatting option disagrees with the expected one true brace style value.

The `indent_brace_style` value is matched case-insensitively. HA0003 reports one diagnostic for each conflicting C# formatting option and includes the effective value it observed. `csharp_prefer_braces` must be `true`; when HA0003 can observe an inline or compiler-provided `IDE0011` severity, that severity must be `error`.

## Motivation

`indent_brace_style` is a general EditorConfig setting, but the .NET formatter uses C#-specific formatting options for brace and newline behavior. HA0003 keeps those settings aligned so a project that declares one true brace style does not silently format C# with conflicting newline rules.

## How to fix violations

When `indent_brace_style` is `1TBS` or `OTBS`, set the matching C# formatting options in `.editorconfig`:

```editorconfig
csharp_prefer_braces = true
dotnet_diagnostic.IDE0011.severity = error

csharp_new_line_before_open_brace = none
csharp_new_line_before_else = false
csharp_new_line_before_catch = false
csharp_new_line_before_finally = false
csharp_new_line_before_members_in_object_initializers = false
csharp_new_line_before_members_in_anonymous_types = false
csharp_new_line_between_query_expression_clauses = false
```

The inline shorthand `csharp_prefer_braces = true:error` is also accepted.

## Code with Diagnostic

```editorconfig
root = true

[*]
indent_brace_style = 1TBS

[*.cs]
csharp_prefer_braces = false
csharp_new_line_before_open_brace = all
csharp_new_line_before_else = true
csharp_new_line_before_catch = true
csharp_new_line_before_finally = true
csharp_new_line_before_members_in_object_initializers = true
csharp_new_line_before_members_in_anonymous_types = true
csharp_new_line_between_query_expression_clauses = true
```

## Code with Fix

HA0003 does not provide a code fix. Update the conflicting `.editorconfig` entries manually to the values listed above.
